Description

Postgress Pool (ppool) database supervisor.

Typically, this is started automatically by adding "ppool" to your application list or using application:start(ppool). However, if you want finer control (for some reason), you can start this manually. If you do so, you can pass the configuration directly to ppool_db_pool_sup:start_link/1.
